# 简介

Advance Monitor 是一个轻量级的前端监控 SDK，帮助开发者快速定位线上问题、分析页面性能、了解用户行为。

## 为什么选择 Advance Monitor

在日常开发中，我们经常会遇到这些问题：

- 用户反馈页面白屏，但本地无法复现
- 某个功能上线后，不知道有没有报错
- 页面加载慢，但不知道慢在哪里
- 想了解用户的操作路径，但没有数据支撑

Advance Monitor 就是为了解决这些问题而生的。

## 核心能力

### 错误监控

自动捕获各类前端错误：

- JavaScript 运行时错误
- Promise 未处理的 rejection
- 资源加载失败（图片、脚本、样式）
- 框架层面的错误（Vue、React）

配合 SourceMap，可以直接定位到源码位置。

### 性能监控

采集 Web Vitals 核心指标：

- **LCP** - 最大内容绘制时间
- **FID** - 首次输入延迟
- **CLS** - 累积布局偏移
- **FCP** - 首次内容绘制
- **TTFB** - 首字节时间

同时支持资源加载瀑布图、长任务分析等。

### 用户行为

记录用户的操作轨迹：

- 页面访问
- 点击事件
- 滚动行为
- 路由变化
- 网络请求

当错误发生时，可以通过面包屑回溯用户的操作路径。

### 数据上报

内置多种优化策略：

- 数据压缩，减少带宽占用
- 敏感信息脱敏
- 重复数据去重
- 请求签名，防止数据篡改
- 离线缓存，断网不丢数据

SDK 核心包压缩后约 **15KB**（gzip），按需加载插件不会增加首屏负担。
