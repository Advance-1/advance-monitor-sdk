# 更新日志

## v1.0.0

发布日期：2024-12

### 新功能

- 错误监控：JS 错误、Promise 异常、资源加载失败
- 性能监控：Web Vitals、资源加载、长任务分析
- 用户行为：点击、滚动、路由变化、网络请求
- 框架集成：Vue 2/3、React ErrorBoundary
- SourceMap：CLI 上传、Vite 插件、自动解析
- 链路追踪：Span/Transaction、前后端打通
- 数据分析：UV/PV 统计、漏斗分析、热力图
- 用户反馈：反馈弹窗、截图功能
- 会话回放：用户操作录制
- 告警通知：邮件、钉钉、飞书、Slack

### 优化

- 数据压缩，减少 70% 带宽
- 敏感信息自动脱敏
- 重复数据去重
- 请求签名防篡改
- 离线缓存，断网不丢数据
